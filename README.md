javafx-persistence
==================

Serves as example for two blog posts:

 * Examples in Branch "jaxb"
* Part 2
 * URL: http://blog.so-geht-software.de/2015/10/javafx-persistence-teil-2-jpa/
 * Examples in Branch "jpa"
